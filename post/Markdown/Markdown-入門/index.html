Markdwon——入門指南

Markdown是一種輕量級標記式語言，創始人為約翰·格魯伯（John Gruber）。它允許人們「使用易讀易寫的純文字格式編寫文件，然後轉換成有效的XHTML（或者HTML）文件」。這種語言吸收了很多在電子郵件中已有的純文字標記的特性。

準備工作-工具

我使用的是 Visual Studio Code ( VSCode ) 來編寫
安裝過程就不闡述了，上網搜一下官網下載就好了

安裝中文包

VSCode 的界面語言默認是英文，不過是有官方的語言包。
打開 VSCode 後，按下左側導航欄的第五個圖標 ( Extensions ) 就是 VSCode 的插件市場 (或者按下 Ctrl + Shit + X)，搜索 “chinese” 就能找到簡體中文跟繁體中文包了。

安裝後之後會提示重啟 VSCode 之後，就能正常顯示了中文界面。 ## 0.2 安裝Markdown增强插件 VSCode 默認是支持 Markdown 的，但是我們還是需要一些額外的插件來輔助，獲得更好的書寫環境。跟上面安裝中文包一樣，在 VSCode 的插件市場中，搜索 Markdown ，安裝 ``Markdown All in One`` 和 ``Markdown Preview Github Styling``。 第一個插件，是一個組合包，優化了 Markdown ， VSCode 必備插件。 第二個插件，是以 Github 使用的 Markdwon 渲染樣式，可以即時本地預覧在 Github Pages 中顯示的效果。
